# Equipo Limite Descripcion
Linterna 4 Proporciona una fuente básica de luz.
Luz Ultravioleta 2 Se utiliza para revelar huellas dactilares y huellas creadas por fantasmas.
Camara de video 4 Crea una transmisión de vídeo, que se transmite al ordenador de la furgoneta. Se usa para detectar orbes fantasmales.
Proyector D.O.T.S. 2 Crea un campo de luces verdes cuando se coloca. Se utiliza para detectar evidencias de proyectores D.O.T.S.
EMF 2 Puede detectar la presencia de campos electromagnéticos en el área. Se utiliza para obtener evidencia de EMF Nivel 5.
Libro de escritura fantasma 2 Se usa para obtener escritura fantasma de un fantasma.
Caja de espíritus 2 Se utiliza para comunicarse con un fantasma a través de preguntas.
Termómetro 2 Muestra la temperatura local de una habitación. Se puede utilizar para detectar temperaturas heladas.
